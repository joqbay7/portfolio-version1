# Portfolio (Vite + TypeScript)

Projeto inicial criado com Vite (vanilla) e TypeScript.

## Pré-requisitos
- Node.js 18+ e npm

## Instalar dependências
```powershell
npm install
```

## Ambiente de desenvolvimento
```powershell
npm run dev
```
Acesse: http://localhost:5173

## Build de produção
```powershell
npm run build
```

## Preview do build
```powershell
npm run preview
```
